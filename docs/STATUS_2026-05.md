## Compiler compatibility status

Fixed and regression-tested:

- GLSL exponent and bitwise operators.
- MilkDrop2 attenuated-band aliases.
- Main-minus-aux subtract mode and aux-first mix ordering.
- `texture_source = sampler_main` reset.
- Scalar texture/warp offsets and scales.
- Scalar-scaled main sample mixes.
- `tex2D`/`tex3D` alias normalization and volume-slice metadata.

Still intentionally bounded:

- Non-uniform main-UV transforms are direct-program-only; controls fallback cannot represent independent X/Y scale.
- Complex shader expressions retain raw source and an explicit controls fallback requirement.
- WebGL volume sampling uses an atlas approximation. Native WebGPU built-in noise uses generated RGBA-compatible sources and native captures are required for exact claims.
